# Visualization Guide - Plotting Simulation Results

The `pauli_distribution` package includes utilities to generate publication-quality plots from simulation results generated by the new simplified API.

## Quick Start

### Method 1: Using the Convenience Helper (Simplest)

After running a simulation and having an output directory with `effective_probs_*.txt` and `running_counts_*.jsonl` files:

```python
from pathlib import Path
from pauli_distribution import plot_simulation_results

output_dir = Path("trial_css_ideal_cnot_10000.0_p_0.003_chosen_seed_100001_20260401_024757")
plots = plot_simulation_results(output_dir, error_bars=True)

# Returns dict with:
# plots['progress'] -> Path to progress plot SVG
# plots['convergence'] -> Path to convergence plot SVG
```

Or from command line:

```bash
cd error_propagation
python -m pauli_distribution.plot_results /path/to/output_directory
```

### Method 2: Direct Function Calls

For more control, call the plotting functions directly:

```python
from pathlib import Path
from pauli_distribution import plot_progress_file, plot_convergence_file

output_dir = Path("trial_output_dir")
progress_file = output_dir / "effective_probs_ideal_20260401_024757.txt"
counts_file = output_dir / "running_counts_ideal_20260401_024757.jsonl"

# Plot convergence metrics
plot_progress_file(
    progress_file=progress_file,
    counts_file=counts_file,
    error_bars=True,
    smooth_window=5,
)

# Plot convergence deltas
plot_convergence_file(
    progress_file=progress_file,
    counts_file=counts_file,
    error_bars=True,
    with_bias_convergence=True,
)
```

### Method 3: Command Line Interface

Both plot utilities have full command-line interfaces:

```bash
# Plot progress metrics with default options
python error_propagation/pauli_distribution/plot_progress_file.py \
  --progress-file /path/to/effective_probs_*.txt

# With error bars, counts-file awareness, and window summaries
python error_propagation/pauli_distribution/plot_progress_file.py \
  --progress-file /path/to/effective_probs_*.txt \
  --counts-file /path/to/running_counts_*.jsonl \
  --error-bars \
  --window-summary

# Plot convergence deltas
python error_propagation/pauli_distribution/plot_convergence_file.py \
  --progress-file /path/to/effective_probs_*.txt \
  --counts-file /path/to/running_counts_*.jsonl \
  --error-bars \
  --with-bias-convergence
```

## Output Files

Both utilities save SVG files (scalable vector graphics) to the same directory as your input progress file:

- `effective_probs_*_plots.svg` - Progress metrics vs iteration/samples
- `effective_probs_*_convergence_plots*.svg` - Convergence deltas vs iteration/samples

These can be opened in any web browser or vector graphics editor, and can be easily embedded in documents or presentations.

## Available Options

### plot_progress_file
- `--smooth-window`: Apply moving average smoothing (default: 1, no smoothing)
- `--bias-log-scale`: Use logarithmic scale for bias axis
- `--error-bars`: Include uncertainty bands from counts file
- `--iter-start`, `--iter-end`: Restrict plot range
- `--x-axis`: Use iteration or sample count on x-axis
- `--window-summary`: Compute statistics on specified windows
- `--quality-check`: Run quality assurance checks
- More options available with `--help`

### plot_convergence_file
- `--smooth-window`: Apply moving average smoothing
- `--log-scale`: Use logarithmic scale
- `--error-bars`: Include uncertainty bands
- `--no-bias-convergence`: Exclude bias convergence plot
- `--x-axis`: Use iteration or sample count on x-axis
- More options available with `--help`

## Integration with New API

The plotting utilities work seamlessly with the simplified `error_propagation_simulation()` API:

```python
import os
import time
from pathlib import Path
from datetime import datetime
from pauli_distribution import (
    error_propagation_simulation,
    plot_simulation_results
)

# Create output directory
timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
output_dir = Path(f"trial_output_{timestamp}")
output_dir.mkdir(exist_ok=True)
os.chdir(output_dir)

# Run simulation
progress_file, counts_file = error_propagation_simulation(
    p_param=0.003,
    system_bias=10000.0,
    qubit_platform="ideal",
    gate_sequence=my_gates,
    samples_per_iteration=100000,
    total_samples=200_000_000,
    chosen_seed=1000001,
    timestamp=timestamp
)

# Immediately plot results
os.chdir("..")  # Back to parent directory
plots = plot_simulation_results(output_dir)
print(f"Progress plot: {plots['progress']}")
print(f"Convergence plot: {plots['convergence']}")
```
